# Channel Specs — Quick Reference

Hard constraints for each format. Check these before delivering.

---

## Blog Post

| Constraint | Value |
|-----------|-------|
| Total length | 800–1500 words |
| Intro | 80–120 words |
| Each body section | 150–300 words |
| Closing section | 100–150 words |
| H1 title | Under 60 characters |
| Meta description | Under 155 characters |
| Internal links | Minimum 1, ideally 2 |
| H2 headings | Statements only — no questions |
| Image placeholder | Required after H1 |
| CTA | Specific — never "I hope this helped" |

**Tone:** Most formal of the three. Public-facing expert content.

---

## Newsletter

| Constraint | Value |
|-----------|-------|
| Total length | 350–500 words (non-negotiable) |
| Opening | 40–60 words |
| Main content | 200–280 words |
| Secondary note | 60–80 words (optional) |
| CTA block | 30–50 words |
| Subject lines | 3 options, each under 45 characters |
| Preview text | 60–90 characters |
| Paragraphs | Max 3 sentences each |
| CTAs | Exactly 1 |
| Headings | None — flowing paragraphs only |
| Formatting | No tables, no complex markdown |

**Tone:** Most personal of the three. One person to one person.

**Subject line angles (must be genuinely different):**
- A: Curiosity (creates open)
- B: Direct benefit (creates click)
- C: Personal or question (creates connection)

## Google Business Profile Post

| Constraint | Value |
|-----------|-------|
| Hard character limit | 1500 characters including spaces |
| Target range | 900–1200 characters |
| Opening hook | 100–150 characters |
| Body | 600–800 characters |
| CTA | 100–150 characters |
| Hashtags | None — do not use |
| URLs | None — phone as plain text only |
| Topics | One only |

**Mobile preview:** First ~100 characters appear before "Prikaži više" tap.
The opening must earn that tap on its own.

**CTA formats that work in GBP:**
- `Zakažite termin: 060 123 4567`
- `Pozovite nas: 060 123 4567`
- `Zakažite na sensiskin.rs`

**Tone:** Local, practical, specific. Mention Novi Sad where natural.

## Format comparison at a glance

| | Blog | Newsletter | GBP |
|--|------|-----------|-----|
| Length | 800–1500 words | 350–500 words | 900–1200 chars |
| Headings | H1 + H2 | None | None |
| Tone | Expert, public | Personal, 1:1 | Local, practical |
| CTAs | 1 (end) | 1 (specific) | 1 (with phone) |
| Links | Internal placeholders | 1 link in CTA | No URLs |
| Images | Placeholder required | Platform-managed | Suggest a photo |
| SEO | Meta description required | N/A | N/A |
