When writing Serbian copy, Serbian declension for skin types is a common failure point.

"Koža" (skin) as a category noun in plural or accusative contexts:
- Nominative singular: "koža" (the skin)
- Accusative singular: "kožu" (for the skin)
- Plural nominative: "kože" (skin types)
- "Za kože sklone aknama" NOT "za koža sklone aknama"
- "Kože s rozaceom" NOT "Rozacea i reaktivne koža"
- In bullet lists: if subject is plural skin types → "kože"

Pattern: when listing types of skin as subjects or objects, always verify plural forms are "kože" not "koža" (genitive plural) unless genitive plural is specifically required.

**Why:** User corrected "za koje koža" → "za koje kože" mid-task, flagging case awareness as a non-negotiable quality bar.

**How to apply:** Before finalizing any Serbian copy that references skin types in plural, do a case-check pass on "koža/kože/kožu/kožom" occurrences. Ask: is this singular nominative, singular accusative, or plural?
